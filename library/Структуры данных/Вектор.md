# Вектор
## Определение
Вектор - это динамический массив, который автоматически изменяет свой размер при добавлении или удалении элементов.
## Структура
```cpp
template<typename T>
class Vector {
    T* data;       // Указатель на начало массива в куче
    size_t size;   // Текущее количество элементов
    size_t capacity; // Общий размер выделенной памяти
};
```
