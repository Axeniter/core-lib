# MVC
MVC - архитектурный паттерн, разделяющий структуру программы на 3 слабосвязанных слоях:
1. Model - содержит бизнес-логику приложения. Может содержать логику приложения, взаимодействие с базами данных и т.д.
2. View - отображение данных для пользователя (представление данных из Model). Может быть в виде UI компонентов, JSON данных и т.д.
3. Controller - движок, обрабатывающий запросы пользователя и обеспечивающий связь между Model и View. Взаимодействует с Model на основе ввода пользователя и возвращает View пользователю.

Паттерн часто используется в backend фреймворках.
