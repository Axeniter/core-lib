# Стек
## Определение
Стек - это структура данных, которая работает по принципу "последний пришёл - первый ушёл" (LIFO, Last In, First Out). Это означает, что последний добавленный элемент будет первым, который будет удалён. Стек можно представить как стопку книг, где вы можете добавлять новые книги только сверху и убирать только верхнюю книгу.
## Операции
- Push - добавление элемента на верх стека.
- Pop - удаление элемента с верхушки стека.
- Peek - получение верхнего элемента стека.
## Реализация
Стек можно реализовать как с помощью односвязного списка, так и с помощью массива.

Реализация на массиве на языке C (без проверки на выход за границу стека):
```c
typedef struct
{
    int *items;
    int head;
} Stack;

void make_stack(Stack *st, int size)
{
    st->items = (int*)malloc(size*sizeof(int));
    st->head = -1;
}

int is_empty(Stack *st)
{
    return st->head == -1;
}

void push(Stack *st, int num)
{
    st->head++;
    st->items[st->head] = num;
}  

void pop(Stack *st)
{
    st->head--;
}

int peak(Stack *st)
{
    return st->items[st->head];
}
```
# Очередь
## Определение
Очередь - это структура данных, которая работает по принципу "первый пришёл - первый ушёл" (FIFO, First In, First Out). Это означает, что первый добавленный элемент будет первым, который будет удалён. Можно представить как очередь из людей в магазине, тот кто первый пришёл, того первого и обслужат.
## Операции
- Enqueue - добавление элемента в конец очереди
- Dequeue - удаление элемента из начала очереди
- Front - получение первого элемента очереди
## Реализация
Очередь можно реализовать как с помощью односвязного списка, так и на обычном массиве.

Реализация на односвязном списке на C:
```C
struct Node
{
    int data;
    struct Node *next;
};

typedef struct
{
    struct Node *first;
    struct Node *last;
} Queue;

Queue* create_queue()
{
    Queue* queue = (Queue*)malloc(sizeof(Queue));
    queue->first = queue->last = NULL;
    return queue;
}

void enqueue(Queue *queue, int element)
{
    struct Node *newElement = (struct Node*)malloc(sizeof(struct Node));
    newElement -> data = element;
    newElement -> next = NULL;
    if (queue->first == NULL)
    {
        queue->first = queue->last = newElement;
        return;
    }
    queue -> last -> next = newElement;
    queue -> last = newElement;
}

void dequeue(Queue *queue)
{
    if (queue -> first == NULL)
    {
        return;
    }
    struct Node *deleted = queue -> first;
    queue -> first = deleted -> next;
    free(deleted);
    if (queue -> first == NULL)
    {
        queue -> last = NULL;
    }
}

int is_empty(Queue *queue)
{
    return queue -> first == NULL;
}

int front(Queue *queue)
{
    if (isEmpty(queue))
    {
        return -1;
    }
    return queue -> first -> data;
}
```
# Дек
## Определение
Дек (от английского "double-ended queue") - это структура данных, которая позволяет добавлять и удалять элементы как с одного, так и с другого конца. Таким образом, дек сочетает в себе свойства как стека (где элементы добавляются и удаляются с одного конца), так и очереди (где элементы добавляются с одного конца и удаляются с другого).
## Операции
- push_front - добавить элемент в начало
- push_back - добавить элемент в конец
- pop_front - удалить элемент с начала
- pop_back - удалить элемент с конца
- first - получить первый элемент
- last - получить последний элемент
