# Таблица производных
## Обозначения
- $C$ - константа
- $x$ - переменная
- $f=f(x)$, $g=g(x)$, $u=u(x)$, $v=v(x)$ - дифференцируемые функции
- $a$ - некоторая постоянная
## Свойства дифференцирования
- $(C)' = 0$ - производная константы
- $(x)'=1$ - производная переменной
- $(af)' = af'$ - вынесение постоянной за знак производной
- $(u \pm v)' = u' \pm v'$ - производная суммы(разности)
- $(uv)' = u'v + uv'$ - производная произведения
- $\left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2}$ - производная частного
- $(f(g(x)))' = f'(g(x)) \cdot g'(x)$ - производная сложной функции
- $y_{x}'=\frac{y'(t)}{x'(t)}$ - производная параметрически заданной функции: $\begin{cases} x=x(t) \\ y=y(t)\end{cases}$, $t$ - параметр
- $(f^{-1})'=\frac{1}{f'}$ - производная обратной функции
- Производная сложной степенной функции:
	- $(u^v)'=u^v\cdot \ln u\cdot v'+v\cdot u^{v-1}\cdot u'$
	- $(u^v)'=u^v\cdot(v\cdot \ln u)'$
## Производные элементарных функций
Базовые:
- $(C)'=0$
- $(x)' = 1$
Степенная:
- $(x^n)' = n \cdot x^{n-1}$
- $\left(\frac{1}{x}\right)' = -\frac{1}{x^2}$
- $(\sqrt{x})' = \frac{1}{2\sqrt{x}}$
Показательная:
- $(a^x)' = a^x \ln a$
- $(e^x)' = e^x$
Логарифмическая:
- $(\log_a x)' = \frac{1}{x \ln a}$
- $(\ln x)' = \frac{1}{x}$
Тригонометрические:
-   $(\sin x)' = \cos x$
-   $(\cos x)' = -\sin x$
-   $(\tan x)' = \frac{1}{\cos^2 x}$
-   $(\cot x)' = -\frac{1}{\sin^2 x}$
Обратные тригонометрические:
-   $(\arcsin x)' = \frac{1}{\sqrt{1-x^2}}$
-   $(\arccos x)' = -\frac{1}{\sqrt{1-x^2}}$
-   $(\arctan x)' = \frac{1}{1+x^2}$
-   $(\text{arccot } x)' = -\frac{1}{1+x^2}$
Гиперболические:
*   $(\sinh x)' = \cosh x$
*   $(\cosh x)' = \sinh x$
*   $(\tanh x)' = \frac{1}{\cosh^2 x}$
*   $(\coth x)' = -\frac{1}{\sinh^2 x}$
