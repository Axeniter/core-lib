# Таблица интегралов
## Свойства неопределенного интеграла
1. $\displaystyle \left(\int f(x)dx\right)'_{x}=f(x)$
2. $\displaystyle d\left( \int f(x)dx \right)=f(x)dx$
3. $\displaystyle \int d(f(x))=f(x)+C$
4. $\displaystyle \int a\cdot f(x)dx=a\cdot \int f(x)dx$
5. $\displaystyle \int(f_{1}(x)+f_{2}(x))dx=\int f_{1}(x)dx+\int f_{2}(x)dx$
## Таблица неопределенных интегралов
1. $\displaystyle \int x^n dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1, \, x > 0)$
2. $\displaystyle \int \frac{dx}{x} = \ln|x| + C, \, x \neq 0$
3. $\displaystyle \int a^x dx = \frac{a^x}{\ln a} + C, \quad 0 < a \neq 1$
4. $\displaystyle \int e^x dx = e^x + C$
5. $\displaystyle \int \sin x \, dx = -\cos x + C$
6. $\displaystyle \int \cos x \, dx = \sin x + C$
7. $\displaystyle \int \frac{dx}{\cos^2 x} = \tan x + C, \quad x \neq \frac{\pi}{2} + \pi n, \, n \in \mathbb{Z}$
8. $\displaystyle \int \frac{dx}{\sin^2 x} = -\cot x + C, \quad x \neq \pi n, \, n \in \mathbb{Z}$
9. $\displaystyle \int \tan x \, dx = -\ln |\cos x| + C, \quad x \neq \frac{\pi}{2} + \pi n, \, n \in \mathbb{Z}$
10. $\displaystyle \int \cot x \, dx = \ln |\sin x| + C, \quad x \neq \pi n, \, n \in \mathbb{Z}$
11. $\displaystyle \int \frac{dx}{\sin x} = \ln \left| \tan \frac{x}{2} \right| + C, \quad x \neq \pi n, \, n \in \mathbb{Z}$
12. $\displaystyle \int \frac{dx}{\cos x} = \ln \left| \tan \left( \frac{x}{2} + \frac{\pi}{4} \right) \right| + C, \quad x \neq \frac{\pi}{2} + \pi n, \, n \in \mathbb{Z}$
13. $\displaystyle \int \frac{dx}{x^2 + a^2} = \frac{1}{a} \arctan \frac{x}{a} + C$
14. $\displaystyle \int \frac{dx}{x^2 - a^2} = \frac{1}{2a} \ln \left| \frac{x - a}{x + a} \right| + C, \quad |x| \neq |a|$
15. $\displaystyle \int \frac{dx}{\sqrt{a^2 - x^2}} = \arcsin \frac{x}{a} + C, \quad |x| < |a|$
16. $\displaystyle \int \frac{dx}{\sqrt{x^2 \pm a^2}} = \ln |x + \sqrt{x^2 \pm a^2}| + C$, в случае "-" $|a|<|x|$
17. $\displaystyle \int \frac{x \, dx}{x^2 \pm a^2} = \frac{1}{2} \ln |x^2 \pm a^2| + C$, в случае "-" $|x|\neq|a|$
18. $\displaystyle \int \frac{x \, dx}{\sqrt{x^2 \pm a^2}} = \sqrt{x^2 \pm a^2} + C$, в случае "-" $|a|<|x|$
19. $\displaystyle \int \frac{x \, dx}{\sqrt{a^2 - x^2}} = -\sqrt{a^2 - x^2} + C, \quad |x| < |a|$
20. $\displaystyle \int \sinh x \, dx = \cosh x + C$
21. $\displaystyle \int \cosh x \, dx = \sinh x + C$
22. $\displaystyle \int \frac{dx}{\sinh^2 x} = -\coth x + C, \quad x \neq 0$
23. $\displaystyle \int \frac{dx}{\cosh^2 x} = \tanh x + C$
