# Уравнения в полных дифференциалах
## Определение
Дифференциалом функции двух переменных является:
$$du = u_x dx + u_y dy$$

Уравнение $M(x,y)dx+N(x,y)dy=0$ является уравнением в полных дифференциалах, если $\exists u=u(x,y)$, такая, что:
$$du=Mdx+Ndy=u_x dx + u_y dy$$$$du = 0 \quad \Rightarrow \quad u(x,y) = С =\text{const}$$ где $u$ - потенциал.

## Теорема
Пусть $D$ - область на плоскости $\mathbb{R}^2$,  
$M, N, M_y, N_x \in C(D)$ - функции непрерывные на $D$.  
Тогда
$$\exists u, \quad du = M dx + N dy \quad \iff \quad M_y = N_x \quad \text{в } D$$

Доказательство:

$(\implies)$  
$u_x = M$, $u_y = N$  
$u_{xy} = M_y$, $u_{yx} = N_x$  
$u_{xy} = u_{yx} \Rightarrow M_y = N_x$

$(\Longleftarrow)$  
$M_y = N_x$  
Ищем $u$ из условия $u_x = M$:
$$u(x,y) = \int M(x,y) dx + \varphi(y)$$
(при интегрировании функции от нескольких переменных получаем неизвестную функцию от другой переменной в качестве константы)
$$u_y(x,y) = \frac{\partial}{\partial y} \left( \int M(x,y) dx + \varphi(y) \right) = N(x,y)$$
$$\varphi'(y) = N(x,y) - \int M_y(x,y) dx$$
Нужно показать, что правая часть зависит только от $y$.  
Проверим, что производная правой части по $x$ равна $0$:
$$\left( N(x,y) - \int M_y(x,y) dx \right)_x = N_x(x,y) - M_y(x,y) = 0$$

Пример.
$(3x^2 + 6xy^2) dx + (6x^2 y + 4y^3) dy = 0$
$M_y = 12xy = N_x$
$u_x = 3x^2 + 6xy^2$  
$u = x^3 + 3x^2 y^2 + \varphi(y)$
$u_y = 6x^2 y + \varphi'(y) = N = 6x^2 y + 4y^3$  
$\varphi'(y) = 4y^3$  
$\varphi(y) = y^4$
Ответ: $x^3 + 3x^2 y^2 + y^4 = C$

## Замечание
Пусть $M(x,y)dx + N(x,y)dy = 0$ - уравнение в полных дифференциалах ($du$).  
Для нахождения потенциала $u$ достаточно выбрать точку $(x_0, y_0)$ и любую кривую, соединяющую её с точкой $(x, y)$, и проинтегрировать дифференциальное выражение вдоль этой кривой.  
Обычно берут ломаную: сначала по $x$, затем по $y$:
$$u(x,y) = \int_{x_0}^x M(s, y_0) ds + \int_{y_0}^y N(x, s) ds$$

Пример.
$e^{-y} dx - (2y + x e^{-y}) dy = 0$
$M_y = -e^{-y} = N_x$
Выбираем $(x_0, y_0) = (0, 0)$
$u(x,y) = \int_0^x e^{0} ds - \int_0^y (2s + x e^{-s}) ds = x - y^2 + x e^{-s} \Big|_0^y = x - y^2 + x e^{-y} - x = x e^{-y} - y^2$
Ответ: $x e^{-y} - y^2 = C$

## Интегрируемые комбинации
Пример:
$$2xy \, dx + (x^2 - 3y^2) dy = 0$$
уравнение в полных дифференциалах

$$y \, d(x^2) + x^2 dy - d(y^3) = 0$$
$$d(x^2 y) - d(y^3) = 0$$
$$d(x^2 y - y^3) = 0$$
Ответ: $x^2 y - y^3 = C$

## Интегрирующий множитель
Интегрирующий множитель для линейного уравнения
   $$y' + p(x)y = q(x) \quad | \quad e^{\int p(x) dx}$$$$\frac{d}{dx} \left( e^{\int p(x) dx} y(x) \right) = q(x) e^{\int p(x) dx}$$
### Общий случай
$M(x,y)dx + N(x,y)dy = 0$ - допустим, не является уравнением в полных дифференциалах.  
Умножаем на интегрирующий множитель $\mu(x,y) \neq 0$:
$$(\mu M)_y = (\mu N)_x$$
дифференциальное уравнение относительно функции $\mu$ (уравнение в частных производных)

### Частный случай: $\mu = \mu(x)$
Пусть $\mu = \mu(x)$. Тогда:
$$\mu M_y = \mu_x N + \mu N_x$$
$$\mu_x N = \mu (M_y - N_x)$$

Если $\frac{M_y - N_x}{N}$ не зависит от $y$, то существует интегрирующий множитель $\mu = \mu(x)$.

Пример.
$(x^{2}+y^{2}+x)dx+ydy=0$
$M_{y}=2y\neq N_{x}=0$
$(\mu(x)\cdot(x^{2}+y^{2}+x))_{y}=(\mu(x)\cdot y)_{x}$
$2y\cdot\mu(x)=\mu'(x)\cdot y$
$\mu'(x)=2\mu(x)$
$\mu(x)=e^{2x}$
Умножаем на интегрирующий множитель
$e^{2x}(x^{2}+y^{2}+x)dx+e^{2x}ydy=0$
$u_{x}=e^{2x}(x^{2}+y^{2}+x)$
$u_{y}=e^{2x}y$
$u=e^{2x} \frac{1}{2}y^{2}+\varphi(x)$
$u_{x}=e^{2x}y^2+\varphi'(x)=e^{2x}(x^{2}+y^{2}+x)$
$\varphi'(x)=e^{2x}(x^2+x)=\frac{1}{2}(x^{2}\cdot e^{2x})'$
$\varphi(x)=\frac{1}{2}x^{2}e^{2x}$
Ответ: $e^{2x}(x^{2}+y^{2})=C$

Пример. (комбинированный метод)
$x(x^{2}+y^{2})dx+y(x^{2}+y^{2})dy+ydx-xdy=0$
$\frac{1}{2}(x^{2}+y^{2})d(x^{2}+y^{2})$ - объединяем первые два слагаемых
$\left( \frac{y}{x^{2}+y^{2}}+x \right)dx+\left( y-\frac{x}{x^{2}+y^{2}} \right)dy=0$
$y+x(x^{2}+y^{2})dx+(y(x^{2}+y^{2})-x)dy=0$

$M_{y}=\frac{x^{2}+y^{2}-2y^{2}}{(x^{2}+y^{2})^{2}}$
$N_{x}=- \frac{x^{2}+y^{2}-2x^{2}}{(x^{2}+y^{2})^{2}}$
$M_{y}=N_{x}$
Получаем уравнение в полных дифференциалах
