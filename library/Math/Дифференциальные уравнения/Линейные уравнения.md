# Линейные уравнения
## Линейные уравнения первого порядка
Общий вид:
$$y' + p(x) \cdot y = q(x)$$
где $p(x)$ и $q(x)$ - известные функции.
- Если $q=0$ - однородное уравнение.
- Если $q \neq 0$ - неоднородное уравнение.

### Линейный дифференциальный оператор
$$L[y] = y' + p(x) \cdot y$$
Обладает свойством линейности:
$$L[c_1 y_1 + c_2 y_2] = c_1 L[y_1] + c_2 L[y_2]$$
### Случай 1: Однородное уравнение ($q=0$)
$$y' + p(x)y = 0$$

Решение:
Умножаем на интегрирующий множитель $e^{\int p(x) dx}$:
$$y' e^{\int p(x) dx} + y p(x) e^{\int p(x) dx} = 0$$
$$\frac{d}{dx}\left( y(x) e^{\int p(x) dx} \right) = 0$$
Общее решение:
$$y(x) = C \exp\left( -\int p(x) dx \right)$$

Пример 1:
$y' + xy = 0$
$y = C e^{-x^2/2}$
### Случай 2: Неоднородное уравнение ($q \neq 0$)
$$y' + p(x)y = q(x)$$
#### Способ 1: Метод вариации постоянной
1. Решаем однородное уравнение: $y_{\text{одн}} = C \exp\left( -\int p(x) dx \right)$
2. Считаем $C$ функцией от $x$: $y = C(x) \exp\left( -\int p(x) dx \right)$
3. Подставляем в исходное уравнение и находим $C'(x)$:
$$C'(x)\exp\left( -\int p(x)dx \right)-C(x)p(x)\exp\left( -\int p(x)dx \right)+p(x)C(x)\exp\left( -\int p(x)dx \right)=q(x)$$
$$C'(x) \exp\left( -\int p(x) dx \right) = q(x)$$
$$C'(x)=q(x)\exp\left( \int p(x)dx \right)$$
4. Интегрируем:
$$C(x) = \int q(x) \exp\left( \int p(x) dx \right) dx$$
5. Получая $C$ можем записать ответ.

Пример 2:
$y' + 3y = x$
Решение: $y = C(x) e^{-3x}$
$C'e^{-3x}+Ce^{-3x}+3Ce^{-3x}=x$
$C'(x) = x e^{3x}$
$C=\int xe^{3x}dx=\frac{1}{3}\int x de^{3x}=\frac{1}{3}\left( xe^{3x}-\frac{1}{3}e^{3x} \right)+K$ - интегрируем по частям
$y=\left( \frac{1}{3}\left( x-\frac{1}{3} \right)e^{3x} + K \right)e^{-3x}=\frac{1}{3}\left( x-\frac{1}{3} \right)+Ke^{-3x}$
$Ke^{-3x}$ - общее решение однородного уравнения
$\frac{1}{3}\left( x-\frac{1}{3} \right)$ - частное решение неоднородного уравнения

#### Замечание
Структура общего решения:
> Общее решение неоднородного уравнения = Частное решение неоднородного + Общее решение однородного

#### Способ 2: Метод подстановки (Бернулли)
Подстановка $y = u \cdot v$:
$$u'v + v'u + p \cdot u \cdot v = q$$
Группируем первое и третье слагаемые:
$$(u' + p u)v + v'u = q$$
Выбираем $u$ так, чтобы $u' + p u = 0$, тогда:
$$\begin{cases}
u' + p u = 0 \\
v' u = q
\end{cases}$$
1. Из первого уравнения находим $u$
2. Из второго находим $v'$, затем $v$
3. $y = u \cdot v$

Пример 3:
$y' + 3y = x$
$y = uv$
$u'v + v'u + 3uv = x$
$\begin{cases} u' + 3u = 0 \\ v'u = x \end{cases}$
$u = e^{-3x}$
$v'e^{-3x}=x$
$v' = x e^{3x}$
$v = \frac{1}{3}\left( x - \frac{1}{3} \right) e^{3x} + C$
$y = uv = \frac{1}{3}\left( x - \frac{1}{3} \right) + C e^{-3x}$

Пример 4:
$xy'+3y=x^{2}$
Однородное:
$xy'+3y=0$
$x \frac{dy}{dx}=-3y$
$\frac{dy}{y}=-\frac{3}{x}dx$
$\int\frac{dy}{y}=-\int\frac{3}{x}dx$
$\ln|yx^3|=\ln|C|$
$y=Cx^{-3}$ - общее решение однородного уравнения

$y=ax^n$
$anx^n+3ax^n=x^{2}$
$n=2$
$5a=1$
$y=\frac{1}{5}x^2$ - частное решение неоднородного уравнения

$y=\frac{1}{5}x^{2}+Cx^{-3}$

Пример 5:
$y'=\frac{1}{2x-y^{2}}$
$\frac{dx}{dy}=2x-y^{2}$
$\frac{dx}{dy}-2x=-y^{2}$
Рассматриваем $x=x(y)$
$x=uv$
$u_{y}'v+uv'_{y}-2uv=-y^{2}$
$u'_{y}-2u=0$
$u=e^{2y}$
$e^{2y}v'_{y}=-y^{2}$
$v'_{y}=-y^{2}e^{-2y}$
$v=-\int y^{2} e^{-2y}dy=\frac{1}{2}e^{-2y}\left( y^{2}+y+\frac{1}{2} \right)+C$
$x=\frac{1}{2}\left( y^{2}+y+\frac{1}{2} \right)+Ce^{2y}$

## Уравнения Бернулли
Общий вид:
$$y' + p(x)y = q(x) y^n, \quad n \neq 0, 1$$

### Способ 1: Замена $z = y^{1-n}$
После замены получается линейное уравнение относительно $z$.

### Способ 2: Подстановка $y = uv$
1. $u'v+uv'+puv=qu^nv^n$
2. $u' + p u = 0$ - находим $u$
3. $u v' = q u^n v^n \quad \Rightarrow \quad v' = q u^{n-1} v^n$ - уравнение с разделяющимися переменными $v'=q_{1}v^n$, где $q_{1}=qu^{n-1}$

Пример 6:
$y'=\frac{4}{x}y+x \sqrt{ y }$ - уравнение Бернулли
$y=uv$
$u'v+uv'=\frac{4}{x}v+x\sqrt{ uv }$
$u'=\frac{4u}{x}$
$u=x^4$

$v'x^4=x \sqrt{ x^4 v }$
Предположим $x>0$
$v'=\frac{1}{x}\sqrt{ v }$
$\frac{dv}{\sqrt{ v }}=\frac{dx}{x}$
$\int\frac{dv}{\sqrt{ v }}=\int\frac{dx}{x}$
$2\sqrt{ v } = \ln x + C$
$v=\frac{1}{4}(\ln x+C)^{2}$

$y=\frac{1}{4}x^{4}(\ln x + C)^{2}$
