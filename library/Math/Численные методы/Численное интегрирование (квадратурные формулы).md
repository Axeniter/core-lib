# Численное интегрирование (квадратурные формулы)
## Определение определенного интеграла
Пусть функция $f(x)$ определена на отрезке $[a, b]$.  
Разобьем отрезок на $n$ частей точками:
$$a = x_0 < x_1 < \dots < x_n = b$$
Обозначим:
- $\Delta x_i = x_i - x_{i-1}$ - длина $i$-го частичного отрезка
- $\xi_i \in [x_{i-1}, x_i)$ - произвольная точка на $i$-м полуинтервале
- $f(\xi_i)$ - значение функции в этой точке

Интегральная сумма:
$$S_n = \sum_{i=1}^n f(\xi_i) \Delta x_i$$

Определенный интеграл:
$$\int_a^b f(x) \, dx = \lim_{\max \Delta x_i \to 0} \sum_{i=1}^n f(\xi_i) \Delta x_i$$

Геометрический смысл: сумма площадей прямоугольников, аппроксимирующих площадь под кривой.

## Квадратурные формулы
Общий вид квадратурной формулы:
$$\int_a^b f(x) \, dx \approx \sum_{i=0}^n \alpha_i y_i, \quad \text{где } y_i = f(x_i)$$

Коэффициенты $\alpha_i$ и узлы $x_i$ выбираются так, чтобы формула была точной для многочленов определенной степени.

Частный случай: формулы Ньютона-Котеса - получены интегрированием интерполяционного многочлена $L_n(x)$ на равномерной сетке.

## Метод прямоугольников
Разбиение отрезка $[a, b]$ на $n$ частей с шагом $h_i = x_i - x_{i-1}$.

### Формула левых прямоугольников
В качестве $\xi_i$ берем левый конец отрезка: $\xi_i = x_{i-1}$
$$\int_a^b f(x) \, dx \approx h_1 y_0 + h_2 y_1 + \dots + h_n y_{n-1}$$

### Формула правых прямоугольников
В качестве $\xi_i$ берем правый конец отрезка: $\xi_i = x_i$
$$\int_a^b f(x) \, dx \approx h_1 y_1 + h_2 y_2 + \dots + h_n y_n$$

### Формула средних прямоугольников
В качестве $\xi_i$ берем середину отрезка:
$$x_{i-\frac{1}{2}} = \frac{x_{i-1} + x_i}{2}$$
$$\int_a^b f(x) \, dx \approx \sum_{i=1}^n f\left(x_{i-\frac{1}{2}}\right) h_i$$

## Метод трапеций
На каждом частичном отрезке $[x_{i-1}, x_i]$ заменяем функцию прямой, проходящей через точки $(x_{i-1}, y_{i-1})$ и $(x_i, y_i)$.  
Площадь трапеции:
$$\delta_i = \frac{y_{i-1} + y_i}{2} h_i, \quad i = 1, 2, \dots, n$$

Общая формула:
$$\int_a^b f(x) \, dx \approx \frac{1}{2} \sum_{i=1}^n h_i (y_{i-1} + y_i)$$

Случай равномерной сетки ($h_i = h = \text{const}$):
$$\int_a^b f(x) \, dx \approx h \left( \frac{y_0 + y_n}{2} + \sum_{i=1}^{n-1} y_i \right)$$

## Метод Симпсона (парабол)
Идея: на каждом сдвоенном отрезке $[x_{i-1}, x_{i+1}]$ заменяем функцию квадратичным интерполяционным многочленом (параболой), проходящим через три точки:
$$M_{i-1}(x_{i-1}, y_{i-1}), \quad M_i(x_i, y_i), \quad M_{i+1}(x_{i+1}, y_{i+1})$$
Интерполяционный многочлен Лагранжа 2-й степени:
$$L_2(x) = \frac{(x-x_i)(x-x_{i+1})}{(x_{i-1}-x_i)(x_{i-1}-x_{i+1})} y_{i-1} + \frac{(x-x_{i-1})(x-x_{i+1})}{(x_i-x_{i-1})(x_i-x_{i+1})} y_i + \frac{(x-x_{i-1})(x-x_i)}{(x_{i+1}-x_{i-1})(x_{i+1}-x_i)} y_{i+1}$$

Интегрирование на равномерной сетке ($h$ - шаг):
$$\int_{x_{i-1}}^{x_{i+1}} L_2(x) \, dx = \frac{1}{2h^2} \int_{x_{i-1}}^{x_{i+1}} \big[ (x-x_i)(x-x_{i+1}) y_{i-1} - 2(x-x_{i-1})(x-x_{i+1}) y_i + (x-x_{i-1})(x-x_i) y_{i+1} \big] dx$$

После интегрирования получаем формулу Симпсона на одном сдвоенном отрезке:
$$\int_{x_{i-1}}^{x_{i+1}} f(x) \, dx \approx \frac{h}{3} (y_{i-1} + 4y_i + y_{i+1})$$

Составная формула Симпсона (для четного числа отрезков $n$):
$$\int_a^b f(x) \, dx \approx \frac{h}{3} \left[ y_0 + 4(y_1 + y_3 + \dots + y_{n-1}) + 2(y_2 + y_4 + \dots + y_{n-2}) + y_n \right]$$

где $n$ - четное число, $h = \frac{b-a}{n}$.

## Сводка формул (равномерная сетка $h=const$)

| Метод                  | Формула                                                                                                                  |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------ |
| Левые прямоугольники   | $\int_a^b f(x)dx \approx h \sum_{i=0}^{n-1} y_i$                                                                         |
| Правые прямоугольники  | $\int_a^b f(x)dx \approx h \sum_{i=1}^{n} y_i$                                                                           |
| Средние прямоугольники | $\int_a^b f(x)dx \approx h \sum_{i=1}^{n} f(x_{i-\frac{1}{2}})$                                                          |
| Трапеции               | $\int_a^b f(x)dx \approx h \left( \frac{y_0 + y_n}{2} + \sum_{i=1}^{n-1} y_i \right)$                                    |
| Симпсон                | $\int_a^b f(x)dx \approx \frac{h}{3} \left[ y_0 + y_n + 4\sum_{i \text{ нечет}} y_i + 2\sum_{i \text{ чет}} y_i \right]$ |
