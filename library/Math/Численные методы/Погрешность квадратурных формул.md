# Погрешность квадратурных формул
## Общее понятие погрешности
$$R = \int_a^b f(x) \, dx - \sum_{i=0}^n d_i y_i$$
$$R = O(h^r), \quad r > 0$$
где $h = \max_i h_i$ - максимальный шаг разбиения.

Погрешность можно представить как сумму локальных погрешностей на каждом отрезке:
$$R = \sum_{i=1}^n r_i, \quad r_i = \int_{x_{i-1}}^{x_i} f(x) \, dx - \sigma_i$$
где $\sigma_i$ - приближение интеграла на $i$-м отрезке по выбранной формуле.

## Оценка погрешности метода средних прямоугольников
На отрезке $[x_{i-1}, x_i]$ разложим функцию в окрестности средней точки $x_{i-\frac{1}{2}} = \frac{x_{i-1} + x_i}{2}$:

$$f(x) = y_{i-\frac{1}{2}} + y'_{i-\frac{1}{2}} \left( x - x_{i-\frac{1}{2}} \right) + \frac{1}{2} y''_{i-\frac{1}{2}} \left( x - x_{i-\frac{1}{2}} \right)^2 + \frac{1}{6} y'''_{i-\frac{1}{2}} \left( x - x_{i-\frac{1}{2}} \right)^3 + O(h_i^4)$$

Интегрирование:
$$\int^{x_{i}}_{x_{i-1}}f(x)dx=y_{i-\frac{1}{2}}|^{x_{i}}_{x_{i-1}}+y'_{i-\frac{1}{2}}\frac{1}{2}\left( x-x_{i-\frac{1}{2}} \right)^2|^{x_{i}}_{x_{i-1}}+y''_{i-\frac{1}{2}} \frac{1}{6}\left( x-x_{i-\frac{1}{2}} \right)^3+\frac{1}{24}y'''_{i-\frac{1}{2}}\left( x-x_{i-\frac{1}{2}} \right)^4+O(h_{i}^5)$$
$$\left( x-x_{i-\frac{1}{2}} \right)|^{x_{i}}_{x_{i-1}}=\left(  \frac{x_{i}-x_{i-1}}{2}  \right)^{3}-\left( \frac{x_{i-1}-x_{i}}{2} \right)^3=\frac{h_{i}^3}{4}$$
$$\int_{x_{i-1}}^{x_i} f(x) \, dx = y_{i-\frac{1}{2}} \cdot h_i + \frac{h_i^3}{24} y''_{i-\frac{1}{2}} + O(h_i^5)$$

*Пояснение:* слагаемые с нечетными степенями $(x - x_{i-\frac{1}{2}})$ при интегрировании дают нуль из-за симметрии пределов.

Формула средних прямоугольников:
$$\delta_i^{\text{пр}} = y_{i-\frac{1}{2}} \cdot h_i$$

Локальная погрешность метода средних прямоугольников:
$$r_i^{\text{пр}} = \int_{x_{i-1}}^{x_i} f(x) \, dx - \delta_i^{\text{пр}} = \frac{h_i^3}{24} y''_{i-\frac{1}{2}} + O(h_i^5) = \frac{h_i^3}{24} y''(\xi_i), \quad \xi_i \in (x_{i-1}, x_i)$$

## Оценка погрешности метода трапеций
Разложение значений в узлах:
$$y_{i-1} = y_{i-\frac{1}{2}} - \frac{h_i}{2} y'_{i-\frac{1}{2}} + \frac{h_i^2}{8} y''_{i-\frac{1}{2}} - \frac{h_i^3}{48} y'''_{i-\frac{1}{2}} + O(h_i^4)$$
$$y_i = y_{i-\frac{1}{2}} + \frac{h_i}{2} y'_{i-\frac{1}{2}} + \frac{h_i^2}{8} y''_{i-\frac{1}{2}} + \frac{h_i^3}{48} y'''_{i-\frac{1}{2}} + O(h_i^4)$$
Формула трапеций на одном отрезке:
$$\delta_i^{\text{тр}} = \frac{y_{i-1} + y_i}{2} h_i = y_{i-\frac{1}{2}} h_i + \frac{h_i^3}{8} y''_{i-\frac{1}{2}} + O(h_i^5)$$
Сравнение с точным значением интеграла:
$$\int_{x_{i-1}}^{x_i} f(x) \, dx = y_{i-\frac{1}{2}} h_i + \frac{h_i^3}{24} y''_{i-\frac{1}{2}} + O(h_i^5)$$
или:
$$\frac{y_{i-1}+y_i}{2}h_i - \frac{h_i^3}{12}y''_{i-\frac{1}{2}} + O(h_i^5)$$
Локальная погрешность метода трапеций:
$$r_i^{\text{тр}} = \int_{x_{i-1}}^{x_i} f(x) \, dx - \delta_i^{\text{тр}} = -\frac{h_i^3}{12} y''_{i-\frac{1}{2}} + O(h_i^5) = -\frac{h_i^3}{12} y''(\eta_i), \quad \eta_i \in (x_{i-1}, x_i)$$

## Комбинирование формул (правило Рунге)
Взвешенное среднее методов прямоугольников и трапеций дает более точную формулу:
$$I \approx \frac{2I_{\text{пр}} + I_{\text{тр}}}{3}$$
Локальная погрешность комбинированной формулы:
$$r_i = \frac{2r_i^{\text{пр}} + r_i^{\text{тр}}}{3} = O(h_i^5)$$
Это один из способов получения формулы Симпсона:
$$\int^b_{a}f(x)dx \approx \frac{h}{6}\left[ y_{0}+4\left( y_{\frac{1}{2}}+y_{\frac{3}{2}}+\dots+y_{n-\frac{1}{2}} \right) + 2(y_{1}+y_{2}+\dots+y_{n-1}) + y_{n} \right]$$

## Глобальные оценки погрешности (для равномерной сетки)
### Метод прямоугольников
$$|R_{\text{пр}}| = \sum_{i=1}^n |r_i^{\text{пр}}| \leq \frac{h^3}{24} \sum_{i=1}^n |y''(\xi_i)| \leq \frac{h^3}{24} \cdot n \cdot M_2 = \frac{h^2}{24} (b-a) M_2 \quad \xi_{i}\in(x_{i-1},x_{i})$$
где $M_2 = \max_{a \leq x \leq b} |f''(x)|$, $h n = b-a$.
### Метод трапеций
$$|R_{\text{тр}}| \leq \frac{(b-a) h^2}{12} M_2$$
### Метод Симпсона
$$|R_{\text{с}}| \leq \frac{(b-a) h^4}{180} M_4$$
где $M_4 = \max_{a \leq x \leq b} |f^{(4)}(x)|$.
