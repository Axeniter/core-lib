# Сплайны
## Теория
### Причина создания
- Полином Лагранжа $L_n(x)$ на всем отрезке $[a,b]$ может давать большие колебания (особенно на равномерной сетке).
- Теорема Вейерштрасса утверждает, что любую непрерывную функцию можно равномерно приблизить многочленами, но скорость сходимости может быть низкой ($f(x)$ $[a,b]$, $\|f(x)-P_{n}(x)\| \to \infty$).
- Контрпример: $f(x) = |x|$ на $[-1,1]$ плохо приближается полиномами (эффект Гиббса для гладкости).

### Идея сплайнов
Сплайны - кусочная интерполяция: отрезок $[a,b]$ разбивается на маленькие части, на каждой части строится свой полином (обычно одинаковой степени), и они стыкуются с заданной гладкостью.

### Преимущества
- Локальность: изменение одного узла влияет только на соседние отрезки.
- Высокая точность: сплайн 3-й степени дает погрешность $O(h^4)$.
- Наилучшее приближение первой производной.
- Свойство минимальности: сплайн минимизирует функционал:
$$\int_a^b (S''(x))^2 \, dx \leq \int_a^b (f''(x))^2 \, dx$$
  (среди всех функций с заданными узловыми значениями).

### Насыщение интерполяционного процесса
Для гладких функций существует предел точности, выше которого нельзя подняться при фиксированной гладкости сплайна. Оптимальным считается сплайн 3-й степени.

### Обозначение
Сплайн обозначается $S(x)$. При интерполяции табличных значений выполняется $S(x_i) = y_i$.

### Ключевое свойство
$$\int_a^b (S''(x))^2 \, dx \leq \int_a^b (f''(x))^2 \, dx$$
Это означает, что сплайн является "наименее изогнутой" функцией среди всех интерполирующих заданные значения.
