# Погрешности
## Виды погрешностей
1. Неустранимая - погрешность в исходных данных, например неточные измерения.
2. Устранимая - погрешность машинной арифметики, например представление чисел с плавающей точкой в компьютере.
3. Погрешность метода - погрешность несовершенства метода решения задачи.

Корректность и устойчивость: 
- Задача называется корректной, если малое изменение входных данных ведет к малому изменению результата. 
- Алгоритм устойчив, если он не накапливает катастрофически ошибки округления.
